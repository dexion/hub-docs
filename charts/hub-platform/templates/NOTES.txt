== Troubleshooting ==

  Pod stuck pending          : kubectl describe pod ...   (likely PVC StorageClass)
  Ingress 404                : kubectl get ingressclass ; check tls.mode
  cert-manager not ready     : kubectl get clusterissuer ; install cert-manager
  Subcharts can't reach each other: ensure release name = '{{ .Release.Name }}' (used in cross-refs)
